# 🍬 Ecommerce de Golosinas

Aplicación web desarrollada con **React** y **Vite** que simula un ecommerce de golosinas. Permite a los usuarios explorar productos, filtrarlos por categoría, agregarlos al carrito y finalizar una compra de forma simulada.

---

## 🚀 Tecnologías utilizadas

- **React**: Librería para la construcción de interfaces de usuario.
- **Vite**: Herramienta de desarrollo rápida para proyectos frontend modernos.

---

## ✨ Funcionalidades principales

- 📦 **Listado de productos**: Visualización de todos los productos disponibles.
- 🔎 **Filtrado por categoría**: Permite ver productos según su tipo.
- 🛒 **Carrito de compras**: Agregar, visualizar y gestionar productos seleccionados.
- 💳 **Finalización de compra (simulada)**: Proceso de checkout sin integración de pagos reales.
